# spring-boot集成dubbo

Spring Boot集成Dubbo，使用Spring Boot 2.0以上版本和JDK1.8。使用yml属性配置dubbo。可用于生产环境。
使用阿里巴巴官方文档配置方法，不依赖其他个人库。
